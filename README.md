# MQ3 Ethanol detector 

The MQ3 is a gas sensor specialized for **alcohol detection**.

It is a chemiresistor, meaning its resistance changes in response to gas exposure.

Commonly used in breathalyzers, alcohol detectors, and DIY safety projects.
